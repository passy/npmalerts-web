The service is completely free and your email will only be used to send notifications about updates to packages you use.

Notification

Sign up above and you'll get an email when a Node package defined in your package.json has a new release.

All versions

Semver version ranges are checked, but you'll get notified either way in case you want to upgrade your package.json.

About

This page was created by a developer tired of checking for new versions of node packages (especially when using GruntJS). Services he found either required manual checks or tried to charge for it.

The service is free because usage is within the free account on Heroku. If this changes I might ask for donations or add ads to cover costs.

npm alerts is hosted on Heroku and uses the MEAN-stack (i.e. MongoDB, ExpressJS, AngularJS, and NodeJS). It also uses Twitter Bootstrap (actually Angular Bootstrap). If you're interested in the code then feel free to check out the sourcecode at GitHub.
